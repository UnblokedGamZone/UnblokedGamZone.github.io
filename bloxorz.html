About Bloxorz

One of the most iconic and addictive puzzle games from the Flash era! Control a 1×1×2 rectangular block (Bloxorz) and roll it across a floating platform made of tiles. Your goal: get the block to fall into the square hole at the end of each level — but be careful, because if you fall off the edge, you restart! With 33 increasingly mind-bending levels, hidden switches, bridges, switches, and split mechanics in later stages, Bloxorz tests your spatial reasoning, patience, and precision like few other browser games ever have.

Original Creator: Damien Clarke
First Released: 2007
Current Version on site: Classic Flash version (via Ruffle emulator)
Added to Unblocked Game Zone: January 2026

Controls:
• Arrow Keys / WASD → Roll the block in that direction
• Spacebar (optional) → Pause / Menu in some versions

Pro Tip: Think several moves ahead — always visualize where the block will end up after each roll. Standing it upright (on its end) is key for tight spaces and switches!

This site hosts the classic Flash version for nostalgic & educational purposes only.
