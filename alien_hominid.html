Run 3

About Run 3

Run 3 is the legendary third installment in the Run series — a fast-paced, gravity-defying endless runner set in abandoned space tunnels. Play as a variety of quirky alien characters, flip gravity, switch between running on walls/ceilings, and try to survive as long as possible in Infinite Mode or complete story-driven levels in Explore Mode. One of the most addictive browser games of all time!

Original Creator: Joseph Cloutier (player_03)
First Released (Mobile as "Run."): December 11, 2013
Browser/Flash Release (as "Run 3"): June 5, 2014
Current Version on site: Classic Flash version (via Ruffle emulator)
Added to Unblocked Game Zone: January 2026

Controls:
• Arrow Keys / WASD → Move & rotate tunnel
• Space / Up Arrow → Jump
• Z → Switch gravity direction (when available)

Pro Tip: Look ahead and plan your path — especially on higher difficulties!

This site hosts the classic Flash version for nostalgic & educational purposes only.
